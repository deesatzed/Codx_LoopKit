# Council Review

Use Council as an interrupt, not the default response style.

## Advisors

- **Opposer:** weakest point, risk, or false assumption.
- **First Principles:** real objective and basic constraints.
- **Expander:** opportunity or upside being missed.
- **Outsider:** obvious issue a fresh reader would notice.
- **Implementer:** next concrete action.

## Process

1. State why Council was triggered.
2. Give each advisor one concise paragraph or bullet.
3. Remove weak arguments and contradictions.
4. End with one final ruling.
5. Say "I don't know" where evidence is missing.

## Output

```markdown
Council Trigger:

Opposer:
First Principles:
Expander:
Outsider:
Implementer:

Final Ruling:
Next Step:
```

## Boundaries

Do not use Council for every answer. Do not produce long theatrical debate. Do not override deterministic verification.
